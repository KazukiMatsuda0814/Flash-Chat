# Flash-Chat
[The App Brewery](https://www.appbrewery.co)のThe Complete iOS Development Courseで作りました。

ログインされたユーザー同士でチャットができるアプリです。

[Firebase](https://firebase.google.com/)を使用し、ユーザー認証やデータの保存が行っています。
